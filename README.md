# stratum
Stratum —— 叙事考古学家  一个运行在终端里的 AI 代理，帮你追溯、解构并预测自己的思想演化。 它不提供建议，只映照你的内心地层。  ✨ 为什么是「叙事考古学家」？  我们每天被海量信息冲刷，却很少停下来问： “我为什么会这样想？这个观念是什么时候进入我大脑的？它又如何一步步变成了现在的样子？”  Stratum 不帮你做决定、不替你写文章、不扮演秘书。 它像一个思想的地质学家，把你的每一次表达、每一个判断、每一段情绪，都视为沉积在地层里的碎片。 你随时可以拿起刷子，挖掘自己过往的对话记录，看清观念的源头与演变。  同时，它还允许你向外考古——丢进一本书、一篇争论、一段历史人物的文字，看它们隐藏的信念与冲突。
🧭 核心能力

自我观念考古

自动提取对话框中的「观念原子」（判断、信念、价值观）
按时间线追溯某个观念的起源、强化、逆转
检测信念冲突，给出思想压力预警
推断多个表层观念背后的根基信念
生成人格侧写、价值观排序、人生故事线
情绪与心理工具

情绪地层图、内心天气、情绪地貌
内在批评家对话、肯定语生成
英雄之旅阶段分析、生命之轮评估
内在小孩、未来自我、反自我人格对话
叙事与创作

梦境分析与续写
思想实验模拟、心理剧剧本
给他人/自己的未寄出的信、时间胶囊
人生座右铭、使命宣言提炼
向外看：文本考古

导入书籍/文章，自动分段提取观念原子
绘制文本的观念演化链（如庄子的“逍遥”观如何变化）
比较两个文本的根基信念差异与冲突点
支持多文本切换，一键切换分析对象
可视化仪表盘
输入 /dashboard，一键启动本地 Web 面板，呈现：

观念原子/演化链数量
情绪波动折线图、正负面情绪分布
高频观念与最新演化链卡片
持续进化
每次对分析结果给出准确/不准确的反馈，Stratum 会记录并用于优化未来的同类输出，越用越懂你。

🛠 技术架构与部署

语言：Python 3.10+
AI 模型：DeepSeek（或其他兼容 OpenAI API 的模型）
存储：SQLite 本地数据库，所有数据永不离机
交互：终端对话，支持 Tab 补全、命令历史
可视化：Flask + Chart.js + Bootstrap，自动启动浏览器面板
打包：PyInstaller 打包成独立 macOS 可执行文件，无需 Python 环境
版本管理：已部署到 GitHub，方便迭代与分享
